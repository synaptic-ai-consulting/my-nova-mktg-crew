### Install Nova — two options

**Option 1: One-click install (recommended)**  
After you have [Cursor](https://cursor.com) installed on your computer:

1. Open: **[https://nova-installer.vercel.app/](https://nova-installer.vercel.app/)**
2. Click **“Install Nova in Cursor — Start here.”**
3. Follow the guided chat in Cursor. It will ask where you want your project folder, then create the files step by step (you approve each step).
4. When it’s done, in Cursor go to **File → Open Folder** and open that project folder. You’re ready to build your crew.

**Option 2: Download the artifacts (GitHub)**  
If you prefer to clone or download the repo and add the files to your project yourself:

1. Go to: **[https://github.com/synaptic-ai-consulting/nova](https://github.com/synaptic-ai-consulting/nova)**
2. Use **Code → Download ZIP** (or clone the repo).
3. Copy the `.cursor` folder and other Nova files into your Cursor workspace. See the repo’s **INSTALL-GUIDE.md** for details.
4. In Cursor chat, type **@wcb-agent.md** and say something like: *“Hi, I’d like to build a work crew.”* Nova will guide you from there.

Both paths get you to the same place: a Nova project where you describe your role, Nova suggests a crew, and you start delegating work.
