## Copy Principles Applied

| Principle | How it shows up |
|---|---|
| **Real, not manufactured** | Price increase is verifiable on the LP. Copy says "you can see it on the course page." No countdown timers or fake scarcity. |
| **Specific numbers** | $1,500 → $1,995 = $495 saved. With LL25: $870 saved. With scholarship: $1,245 saved. Specificity builds trust. |
| **Carmelo's voice** | First person, direct, conversational. "I wanted to reach out" not "We're reaching out." No corporate-speak. |
| **Loss framing** | "That's $495 you keep" and "waiting costs more — literally $495 more" — frames as a cost of inaction, not a discount. |
| **No stacking pressure** | Each block uses only the levers appropriate for that audience. LL25 emails don't mention scholarships. Scholarship emails don't push LL25. Clean segmentation. |
| **Honest close** | Last-day version ends with "whether you join or not" — respects the reader's decision. |
