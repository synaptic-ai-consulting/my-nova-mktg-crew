## 2. Gold Offer (Current Flagship)

### 2.1 Positioning

- **Tagline:** *Software development is moving from writing code to orchestrating agents. This course is what you need to stay relevant—and protect your career.*
- **Value prop:** Project-based course using CrewAI and AAMAD to design, build, and deploy a **production-ready multi-agent AI application** with a real Capstone Project and live coaching.
- **Audience:** Software developers, tech leads, AI/ML engineers (5+ years experience, intermediate Python).
- **Differentiator:** No empty theory; coordination and context-engineering discipline; portfolio-ready Capstone.

### 2.2 Structure (6 weeks)

| Week | Theme | Key Modules | Deliverables |
|------|-------|-------------|--------------|
| Pre-course | Onboarding | Welcome, AAMAD articles, dev setup, Agentic Architect Fundamentals | Dev env setup; Certificate; Goals reflection |
| 1 | Define | Use case selection; Define (Scope, Context, PRD) | MRD; PRD |
| 2 | Build (Architecture + Frontend) | Solution Architecture; Frontend | SAD; Frontend code |
| 3 | Build (Backend) | CrewAI Crew: Agents, Tasks, Tools, Flows | Backend code |
| 4 | Build (Integration) | Full-stack integration; HITL; APIs | Integrated code |
| 5 | Build (Quality + Deploy) | Observability; QA; CI/CD; Hosting | Final code; Deployed app |
| 6 | Deliver | Present; Wall of Fame | Capstone published |

### 2.3 Included

- Live sessions (1 hr/week + 90 min weekly)
- 1:1 coaching
- Lifetime access
- Certificate of completion
- Community of peers
- Maven Guarantee (refund through week 2)

## 3. Silver Offer (Agentic Architect 101 — Live Cohort)

### 3.1 Positioning

- **Tagline:** *Same AAMAD content as the free course—but live, with me. Get unstuck, get reviewed, get it done.*
- **Value prop:** The **Learn AAMAD** curriculum (Define–Build–Deliver mini-project) delivered in **4 hours of live sessions** with **live coaching**: debugging, architecture review, and help translating examples to your real use case.
- **Audience:** Bronze completers (or those who've started) who want accountability, feedback, and live support. Same ICP as Gold but lower commitment and price.
- **Funnel role:** Bridge between Bronze (free) and Gold (full Capstone). Tier 1 on [Learn AAMAD](https://synaptic-ai-consulting.github.io/learn-aamad/).

### 3.2 Content Scope: Same as Bronze (Learn AAMAD)

Silver covers the **same modules** as [Learn AAMAD](https://synaptic-ai-consulting.github.io/learn-aamad/), but live:

| Bronze (self-paced) | Silver (live) |
|---------------------|---------------|
| 01. Introduction to Agentic Architect | Live Session 1 |
| 02. AAMAD Overview | Live Session 1 |
| 03. Context Engineering Basics | Live Session 1 |
| 04. Building Multi-Agent AI Systems with CrewAI | Live Session 2 |
| 05. Hands-On: Define Phase | Live Session 2 |
| 06. Hands-On: Build Phase | Live Session 2 |
| 07. Hands-On: Deliver Phase | Live Session 2 |
| 08. Value and Next Steps | Live Session 2 |

**Total:** 4 hours live (flexible format; see 3.4).

### 3.3 Silver Value-Add (What You Provide Live)

| Bronze obstacle (from survey) | Silver value-add |
|-------------------------------|------------------|
| Not having anyone to review my decisions/architecture | Live architecture review; hot seats |
| Translating examples to my real use case | Live coaching to apply to your use case |
| Getting unstuck when I hit implementation issues | Live debugging |
| Staying accountable and finishing the material | Cohort + live sessions |
| Time / energy | Structured 4-hour block with you |

### 3.4 Silver Format

| Attribute | Value |
|-----------|-------|
| **Schedule** | 2× 2-hour sessions on Saturdays, two consecutive weekends |
| **Rationale** | No conflict with Gold (Mon + Fri live sessions); keeps calendar clean |

*Initial offer uses this fixed format. Weekday options may be added later if demand justifies.*

### 3.5 Silver Syllabus (4 Hours Live)

#### Pre-course (required)

- Complete [Learn AAMAD](https://synaptic-ai-consulting.github.io/learn-aamad/) (Bronze) or be in progress
- Share certificate or confirm completion before first live session

#### Live Session 1 (≈2 hours)

- Why Agentic Architect; AAMAD in 3 phases; Context Engineering; Multi-agent systems intro
- Q&A; hot seats for those who've started the mini-project

#### Live Session 2 (≈2 hours)

- Hands-on: Define, Build, Deliver (mini-project walkthrough)
- Live debugging; architecture review; translating to real use cases
- Value and next steps; upgrade path to Gold

### 3.6 Silver Pricing & Upgrade Path

- **Price:** $399
- **Upgrade path:** 100% of Silver enrollment fee applies as credit toward Gold enrollment.
- **Credit terms:** Applies only to the **next available Gold cohort** within **4 weeks** of Silver end. Keeps momentum and time pressure; credit does not roll over.

### 3.7 Survey Insights (Feb 2026)

*Source: `course_interest_survey_responses (2).csv` — 9 responses, ~6–7 usable.*

| Insight | Implication |
|---------|-------------|
| **$399 feels high** (3/7) | Emphasize value: live debugging, architecture review = high-touch coaching. Consider early-bird or payment plan if needed. |
